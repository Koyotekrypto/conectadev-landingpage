# Imagens para o efeito Compare (antes/depois)

Coloque aqui **duas imagens** com os nomes:

- **`antes.png`** — visual “antes” (ou use `antes.jpg` se preferir)
- **`depois.png`** — visual “depois” (ou use `depois.jpg`)

O modal do portfólio usa essas imagens para o efeito de comparação com o slider.  
**Usar JPG?** Altere em `src/components/sections/Portfolio.tsx` a constante `COMPARE_IMAGE_EXT` para `'.jpg'` e nomeie os arquivos `antes.jpg` e `depois.jpg`.
